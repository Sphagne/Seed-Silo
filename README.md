Here are my MineCraft seed banks, which I used to find seeds for the community

Here is how it works:

All the files start with a number, say 401

Then it is followed by a "-" or "~"

For Cubiomes session files, it's "-"

For plain text files, it's "~"

In these files  I have collected some seeds from sessions files

After that comes the file description, which is a brief description of the content

If the description starts with "+++", then it's a seed bank

Seed banks are the Cubiome sessions that contain lots of raw seeds with similar general conditions that make a foundation for other sessions to tap into and extract specialized seeds from

All the files that search within a seed bank begin with the same number as their source bank

For instance the file [401 - +++ all structures in 2k] contains lots of seeds with the condition to have all these structures within a 2k radius:
Woodland Mansion,
Desert Pyramid,
Jungle Temple,
Swamp Hut,
Ocean Monument,
Igloo,
Pillager Outpost,
Ancient City

And all these seeds start inside a plains village

This condition is cool, because it's fast and narrows the search to starting in a village, and plains is a medium temperature biome so that there would be more chance to find both hot and cold biomes around, a good prerequisite for the rest of the conditions

Currently there are more than 55000 seeds in that bank, and other sessions can make it their source of search and only iterate through the seeds from that collection for a fast search inside the seeds with all the structures within a 2k radius

For instance the session file [401 - flat plains village peak mansion] would search through these seeds for specialized seeds that have these conditions:

A village within 192 blocks, (we know that the source bank begins inside a plains village), but this new village would be the anchor point for the next conditions

The village should be mostly surrounded by plains, with nearby mountain peaks. and a woodland mansion not too far away

This specialized search would be very fast and result in finding such village with the additional conditions of the source bank, i.e. all the structures within 2k radius and starting within a plains village

This rule applies to all the other source banks within these files and all the other specialized searched within those banks

Edit: By the way, I have removed the directory of the source file, from within the session files, because in each computer it's different, so your session files would not find their source and if you want to continue their search, then you have to manually find the source file yourself and make it the source for the next searches.
